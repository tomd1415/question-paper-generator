Computing GCSE - Year 10 Revision

1. What is the main purpose of a network switch? (2 marks)

  1. To connect multiple networks together
  2. To route data packets between devices
  3. To manage and direct traffic between devices in the same network
  4. To encrypt data sent over the network
Correct Answer:
To manage and direct traffic between devices in the same network

2. Convert the binary number 101011 to decimal. (2 marks)

Correct Answer:

3. Which layer of the TCP/IP model is responsible for establishing connections and ensuring data is sent reliably? (1 mark)

  1. Application layer
  2. Transport layer
  3. Network layer
  4. Data link layer
Correct Answer:
Transport layer

4. Explain two differences between a solid-state drive (SSD) and a hard disk drive (HDD). (4 marks)

Correct Answer:
1. SSDs use flash memory and have no moving parts, making them faster and more durable. 2. HDDs use spinning magnetic disks, which are slower but typically have a larger storage capacity for the cost.

Correct Answer:

6. Name the hardware device used to convert digital signals into analogue signals for transmission over traditional telephone lines. (1 mark)

Correct Answer:
Modem

7. Identify the protocol used to securely transfer files over the internet. (1 mark)

  1. FTP
  2. HTTPS
  3. SMTP
  4. SSH
Correct Answer:
SSH

8. Describe the purpose of the CPU in a computer system. (3 marks)

Correct Answer:
The CPU (Central Processing Unit) is responsible for executing instructions from programs, performing calculations, and managing data flow within the computer.

Correct Answer:

10. Which topology connects all devices to a single central hub? (1 mark)

  1. Bus
  2. Star
  3. Ring
  4. Mesh
Correct Answer:
Star

11. State two advantages of using a cloud storage service. (2 marks)

Correct Answer:
1. Data can be accessed from anywhere with an internet connection. 2. It reduces the need for local storage and backups.

12. How many bits are there in an IPv4 address? (1 mark)

Correct Answer:
32

13. Explain why lossy compression is often used for images and videos. (3 marks)

Correct Answer:
Lossy compression reduces file size significantly by removing unnecessary data, which is acceptable for images and videos as small losses in quality are often unnoticeable to the human eye.

14. What is the ASCII value for the letter 'A'? (1 mark)

Correct Answer:
65

if temperature > 30: print('Hot') else: print('Cold')
Correct Answer:
if temperature > 30: print('Hot') else: print('Cold')

16. What is the purpose of a firewall in a computer network? (2 marks)

Correct Answer:
A firewall monitors and controls incoming and outgoing network traffic based on security rules, helping to protect the network from unauthorised access.

17. State two differences between open-source and proprietary software. (4 marks)

Correct Answer:
1. Open-source software allows users to view, modify, and distribute the source code, while proprietary software does not. 2. Open-source software is often free, whereas proprietary software usually requires a paid licence.

18. Calculate the file size of an image with a resolution of 800x600 pixels and 24-bit colour depth. Give your answer in megabytes (MB). (3 marks)

Correct Answer:
File size = (800 x 600 x 24) / 8 / 1,048,576 = 1.37 MB

19. What does DNS stand for, and what is its primary function? (2 marks)

Correct Answer:
DNS stands for Domain Name System. Its primary function is to translate human-readable domain names into IP addresses.

20. Explain one advantage and one disadvantage of using a ring network topology. (4 marks)

Correct Answer:
Advantage: Data is transmitted quickly as it travels in one direction around the ring. Disadvantage: If one device or connection fails, the entire network can be disrupted.
